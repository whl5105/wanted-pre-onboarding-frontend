## ⚙️ 프로젝트의 실행 방법

> 💡주의 : 해당 프로젝트의 API URL .env에서 REACT_APP_BASE_URL로 설정되어 로컬에서 정상 동작되지 않을 수 있습니다.

**Install**
```bash
npm install
or
yarn install
```

**Build**
```bash
npm run Build
or
yarn Build
```

**Start**
```bash
npm start
or
yarn start
```
